Create Query

Dear Individual/Corporate Client,
You can submit your query on this page. There are different fields which you need to fill up in order to query.

1) Your Query Type: General for General Queries. Case Specific to query about a case which is already filled.You need to enter case ID field for this.
2) Query Category: Should mention your category of query, which may be civil, corporate, criminal etc.
3) There is a subcategory which will change according to the category chosen. Different Categories have different subcategories. So choose wisely.
4) CaseID(optional): If u have chosen query type as Case Specific then this option will appear and will give you options on case IDs already filled by you.
5) Query Brief: In this part you can give a brief description about your query if you want to add any more data.
